Detects nonterminals where there is a conflict between the inferred or explicit type of the nonterminal and an alternative

Helps in situations like 

A: T1 = {...}
B: T2 = {...}
// T1 != T2
X = {
    A,
    B,
}
Y: A = {
    A,
    B
}
Where it will flag the second alternative in both X and Y with an "incorrect type" error.
